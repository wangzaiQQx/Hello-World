# Git教程
```powershell
git init # 初始化
git status # 查看状态
git clone xx@xx.com:xx/xx.git # 克隆32位密钥
git checkout -b xx # 创建分支
git checkout - # 切换到上一次提交
git checkout xx # 切换分支
git branch -d xx # 删除分支
git add . #添加文件
git add xx # 添加单个文件
git commit -m "xx" # 提交
git commit -am "xx" # 提交
git merge --no-ff xx # 合并分支
git reflog # 查看提交记录
git rebase -i head~3 # 合并多个提交
git push #推送 
```
